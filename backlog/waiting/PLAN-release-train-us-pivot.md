# PLAN-release-train-us-pivot

## Goal

Create a predictable release train for the US pivot with phase gates aligned to product, AI quality, and reliability metrics.

## Planned Sequence

1. Sprint 1: US foundation localization (completed)
2. Sprint 2: Taxonomy pack model + APIs (in progress)
3. Sprint 3: Pipeline orchestration
4. Sprint 4: Pipeline workspace UI
5. Sprint 5: Workers' Comp pack + eval gates
6. Sprint 6: Personal Injury pack + pilot readiness

## Gate Criteria

- Product gate: scoped features complete
- Quality gate: tests and eval thresholds pass
- Trust gate: approvals and auditability verified
